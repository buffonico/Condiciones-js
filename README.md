## Triple pagina de condiciones

Proyecto condiciones con una imagen, sticker, y password para el desafio Condiciones.

# Acceso
    https://github.com/buffonico/Condiciones-js
    https://buffonico.github.io/Condiciones-js/


#  Descripción

Esta pagina consta de 3 html con un JS de base, nos muestra como dar borde a una iamgen, como sacar un total de Sticker con un maximo de 10 y finalmente como acceder a una web con 2 pass correctas.
